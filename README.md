## The Problem

Millions of people want to read the Bhagavad Gita. Most never finish it.

Not because they lack interest — but because:

- Traditional books feel intimidating
- Sanskrit is difficult for beginners
- Every app or organization presents **one** interpretation and calls it the truth
- Existing apps feel like PDFs, not experiences
- There's no habit loop, no reason to come back tomorrow

The deeper issue: ISKCON presents one perspective. Sant Dnyaneshwar presents another. Tilak had his own. People pick one, get confused or disconnected, and stop.

**Nobody shows you all of them.**

## What Dnyan Does Differently

### 🔍 Every Shloka, Every Perspective

For each of the 700 shlokas, you see:

- Original Sanskrit
- Transliteration
- Simple meaning
- ISKCON commentary
- Sant Dnyaneshwar's perspective
- Other traditional commentaries

You decide what resonates. No single school of thought is forced on you.

### 🎯 Daily Learning Goals

Set a daily target — say, 5 shlokas. Dnyan tracks your progress, builds your streak, and rewards your consistency with badges when you hit your goal.

This is how you actually finish the Gita.

### 🏆 Badges That Mean Something

| Badge | How to Earn |
|-------|-------------|
| **Arjuna Apprentice** | Complete your first chapter |
| **Karma Yogi** | 7-day streak |
| **Gyani** | Read 100 shlokas |
| **Bhakta** | 30-day streak |
| **Chapter Master** | Complete any chapter with all perspectives read |
| **Dnyaneshwar** | Complete all 18 chapters |

### 📬 Shloka Postcards

Found a shloka that hit differently? Generate a beautiful shareable card — your chosen verse, your chosen meaning, minimal mandala-inspired design. Made for Instagram, made for your story, made for the moment.

This is how Gen Z actually shares what moves them.

### 🟩 GitHub-Style Progress Calendar

See your entire learning journey as a contribution graph. Every day you read — a square fills. Watch your spiritual consistency become something visual, something real.

## Tech Stack

| Layer | Tool |
|-------|------|
| Frontend | React + TypeScript + Vite |
| Styling | Tailwind CSS |
| State | Zustand |
| Storage | IndexedDB + LocalStorage |
| PWA | Service Workers + Offline Caching |
| UI Design | Google Stitch |
| AI Coding | Google Antigravity |
| Deploy | Vercel / Cloudflare Pages |

> **On AI explanations:** Dnyan intentionally avoids AI-generated commentaries to prevent hallucinations on sacred text. All perspectives come from actual traditional sources and verified translations.

## Data

All 18 chapters and 700 shlokas are sourced from open-source Bhagavad Gita repositories in structured JSON format. Data is bundled locally within the app — no API dependency, no rate limits, full offline support.

---

## Design Philosophy

Dnyan's visual identity draws from **mandala geometry** — the sacred, symmetrical patterns that have represented cosmic order in Indian tradition for centuries. Warm saffron and ochre tones. Clean typography. Nothing that feels like a generic spiritual app.

The goal is a reading experience that feels as intentional as what you're reading.

## Contributing

Dnyan is open source and community-driven — just like Sant Dnyaneshwar's original mission.

You can contribute:

- **Translations** — help bring Dnyan to more languages
- **Commentaries** — verified traditional perspectives we're missing
- **UI/UX** — design improvements and accessibility fixes
- **Features** — pick an issue, build it
- **Documentation** — help others get started

```bash
git clone https://github.com/Kaustubhhbhoirr/Dnyan.git
cd Dnyan
npm install
npm run dev
```
